## 声明
```CSharp
public struct CrownRotate360Data
```

表冠归一化旋转信息数据结构，表冠旋转角度在[-360,360]范围内，初始旋转角度为0度。

变量名 | 类型| 描述

 MaxAngle  | int | 表冠旋转最大角度
 MinAngle  | int | 表冠旋转最小角度
